# Anes Portfolio (Flat Root Structure)

This portfolio is fully flattened.

## Important
- No subfolders are required.
- Main entry is `index.html`.
- All demo projects are root-level files like:
  - `sttrop-index.html`
  - `hotel-index.html`
  - `restaurant-demo-index.html`
  - `cafeteria-index.html`
  - `rentcar-index.html`

## GitHub Pages deploy for `anesm-dev.github.io`
1. Create a GitHub repository named exactly `anesm-dev.github.io`.
2. Copy all files from this folder into that repo root.
3. Commit and push to `main`.
4. In GitHub: `Settings -> Pages`.
5. Choose `Deploy from a branch`, then `main` and `/ (root)`.

Live URL:
- `https://anesm-dev.github.io/`
